在 VPS 上搭建 Shadowsocks 服务

发表于 2017-11-12 本文字数： 4.3k 阅读时长 ≈ 4 分钟

因为一系列的原因，中国大陆政府逐渐建造起了一堵网络防火墙（Great Firewall of China，GFW），它会通过各种技术手段干扰甚至阻止大陆人民访问被列入黑名单的网站的服务器。然而 ClowWindy 大神开发了一套可以穿透 GFW 的强大工具，它就是 Shadowsocks。

这篇文章记录了我在 VPS 上搭建 Shadowsocks 服务的过程。

原理

通常来说，我们要访问某个网站，只需直接向其服务器发送请求，然后服务器返回给我们数据。由于 GFW 的阻挡，我们不能直接访问某些服务器，然而 Shadowsocks 能使数据突破 GFW 的封锁，与被屏蔽的服务器交互。它的大体原理如下：

  • Client：我们电脑、手机上要访问网络的应用程序
  • SS Local：我们电脑、手机上安装的 Shadowsocks 客户端软件
  • SS Remote：我们购买的服务器上的 Shadowsocks 服务器端软件
  • Target：我们要访问网站的服务器

通过 Shadowsocks 的请求过程：

  1. Client 使用标准 Socks5 协议发送数据给 SS Local
  2. SS Local 使用变种 Socks5 协议发送数据给 SS Remote，此过程将通过 GFW，数据被加密传输
  3. SS Remote 拿到数据后将其解密，发送请求给 Target
  4. Target 发送响应数据给 SS Remote，数据以相同的方式反向传回 Client

由此可见，SS Remote 其实是起到了代理服务器的功能，这就要求它必须在能顺利访问在黑名单上的网站的服务器的地区。

购买 VPS

以上提到，需要一台位于 GFW 之外的服务器，而 VPS（Virtual Private Server）相比真正的独立服务器来说要便宜很多了，是很多人的首选，所以我也决定租一台 VPS 来用。我选择的是 Vultr 每月 5 美元的 VPS。

% note info %}

有许多 VPS 提供商，例如 Lionde、Vultr、BandwagonHost 等，Lionde 最低每月 5 美元，Vultr 最低也是每月 5 美元（原来有每月 2.5 美元的，但是我买的时候暂时没货了），BandwagonHost 太乱没看。我原先想买 Lionde 的，但是只能用信用卡或 PayPal，我为此还申请了一张信用卡，还没邮寄到，才发现 Vultr 支持支付宝，果断下单了，就是每月 30 多块钱，肉疼啊，双十一刚剁完手😑。

% endnote %}

Vultr 的注册与充值都很简单，添加 VPS 也很简单，需要选择 Server Location（位置）、Server Type（操作系统）、Server Size（配置）。

配置 Shadowsocks

实际上现在已经可以使用 ssserver 命令启动 Shadowsocks 服务了，但是这样的话每次要带一大堆参数，指定端口号、加密方式那些，太麻烦了，因此我选择在配置文件中指定这些参数的方式。

首先在 /etc/ 下新建一个 shadowsocks.json 文件，然后使用 Vim 编辑：

我不是很熟悉 Vim，然而在 VPS 上还是老老实实用它吧，真蛋疼。

配置文件里有这几个参数可用：

参数名 意义
server 服务器 IP 地址
server_port 服务器中 Shadowsocks 服务端的端口号
local_address 本地 IP 地址（一般都是 127.0.0.1）
local_port 本地 Shadowsocks 客户端的端口号
password 用于加密数据的密码
timeout 超时时间（秒）
method 加密方法（默认 aes-256-cfb）
fast_open 是（true）否（false）使用 TCP_FASTOPEN
workers Workers 的数量（仅在 Unix 下可用）
